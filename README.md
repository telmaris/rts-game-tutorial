

gra RTS

założenia:

    - mapa 2D (wyrenderowana w 2D LUB izometrycznym 3D)
    - gra real time (RTS) czyli wszystko dzieje się w czasie rzeczywistym!!!
    - gra polega na budowaniu budynków produkujących surowce/towary, magazynowaniu, zaopatrzeniu + system walki autobattle (bardziej ekonomia/logistyka niż walka)
    - duża ilość produktów, nacisk na zoptymalizowanie tej uproszczonej ekonomii
    - łańcuch logistyczny: wydobycie - transport - magazynowanie -* przetwórstwo - magazynowanie
    - system twierdz wytyczających terytorium, utrata twierdzy wiąże sie z utratą budyków i surowców w terytorium
    - autobattle wykorzystujący logistykę zasobów (zaopatrzenie), położenie (geometria mapy), taktykę (np. mechanizm wsparcia itp), pogodę/czas, zadarzenia losowe
    -* opcjonalnie system nacji z bonusami
    - produkcja w czasie rzeczywistym: każdy budynek produkcyjny jak i transport, musi cały czas obliczać output
    - system transportu materiałów: budynek produkcyjny wybiera cel transportu; czas transportu uzależniony od odległości
    - budynek typu splitter (czyli: target budynka produkcyjnego, który rozdziela towar na części i rozprowadza w kilka miejsc) - ALBO opcja transportu do kilku targetów
